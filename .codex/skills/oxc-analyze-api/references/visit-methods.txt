# OXC Visit trait — CURATED EXTRACT for Svelte analyzer
# Source: https://raw.githubusercontent.com/oxc-project/oxc/refs/heads/main/crates/oxc_ast_visit/src/generated/visit.rs
# Updated: 2026-03-28
#
# This is a curated subset containing only methods useful for a Svelte compiler analyzer.
#
# REMOVED:
# - All TypeScript (ts_*) methods — TS is stripped during preprocessing
# - All JSX methods — Svelte doesn't use JSX
# - All JSDoc methods — not applicable
# - Collection visitors (visit_*s methods like visit_statements, visit_expressions) — just iterate
# - visit_span — not needed for analysis
#
